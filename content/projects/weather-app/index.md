A conversational AI chatbot for a grocery business, built during a 24-hour hackathon sprint. The bot handles product queries, order assistance, and FAQs using Rasa NLU + a FastAPI backend.

## Overview

Built during the **AI Conclave Hackathon**, the system needed to handle realistic grocery customer queries with intent classification, entity extraction, and business logic integration — all within a 24-hour window.

## Key Features

- **Intent Recognition** — Rasa NLU trained on grocery-specific intents (product search, pricing, availability, order status)
- **Slot Filling** — Entity extraction for product names, quantities, and locations
- **FastAPI Backend** — REST webhook server integrating Rasa with business logic
- **Custom Actions** — Python action server for dynamic responses (inventory lookup, pricing)

## Tech Stack

| Layer | Technology |
|---|---|
| NLU / Dialogue | Rasa |
| API Backend | FastAPI |
| Language | Python |
